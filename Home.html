University Statistics

Background Information

This is a website which provides Information about the United Kingdom and other countries by the number of students enrolled in higher education courses. The source for the figures is the World Bank Data http://www.worldbank.org/ and Github https://github.com/Hipo/university-domains-list used as API to find information between 2009/15. The student numbers include full-time and part-time, and are undergraduate and postgraduate students. This is to engage the development community with real-world statistics.
